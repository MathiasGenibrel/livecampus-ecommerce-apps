## How to launch apps ?

To launch the React application, use the following command
```bash
nx serve apps-web-app

# Or use npx

npx nx serve apps-web-api
```

To launch the Express application, use the following command
```bash
nx serve apps-web-api

# Or use npx

npx nx serve apps-web-api
```
